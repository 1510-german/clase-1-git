Como inicializar un proyecto con GIT

  1. Crear un repositorio GIT

    git init

  2. Agregar archivos al repositorio (tomar un snapshot - foto del codigo)

    git commit -m "mensaje del commit"

  3. ver el historial de cambios - podemos utilizar el plugin de VS CODE Gitgraph

    git log

  4. Modificar nombre de la branch (rama)

    git branch -m nombre-rama

  5. vincular mi proyecto git con un repositorio en github

    git remote add origin HTTP://url_repositorio.git

  6. subir mi proyecto a hithub

    git push -u origin nombre-rama
